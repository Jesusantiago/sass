Aprendiendo SASS

Primera clase de SASS - Sintaxis del SASS - SCSS

Primero que todo debemos debemos de saber que hay dos maneras de poder crear un archivo SASS, la primera y mas antigüa es con .sass y la que nosotros trabajaremos y es la más actual es la de .scss

Ahora que ya sabemos esto, es importante saber que en este modo de extención se utilizan casi la misma sintaxis que en CSS PURO, es decir por lo momentos sabremos que se deben utilizar las llaves { } para avisar donde comienza y termina las declaraciones que tienen cada una de los elementos con que estemos trabajando.
También debemos de saber que una forma de ingresar a elementos que estan dentro de otros elementos con el CSS PURO, es llamando al elemento padre y luego llamando al elemento hijo cuantas veces sea necesario hasta poder llegar al elemento que nosotros querramos modificar. Esto con SASS no es necesario, con SASS podremos hacer uso del scope del padre para incluir ahí los elementos que querramos tocar. Y lo mas genial de esto que es dento del mismo scope del que estamos trabajando podemos trabajar con pseudoelementos y estos se declaran de la siguiente manera
&:hover{}

Como conectar tu hoja de SASS con el HTML

  1. Debemos de ingresar a nuestra consola de comando
  2. Dirigirnos a la carpeta donde tenemos el proyectos en el que estemos trabajando
  3. Una vez que ya estemos en la carpeta que tenga los archivos de SASS y el CSS que querramos, nosotros debemos ejecutar en la consola de comandos la siguiente linea de código: sass nombreDelArchivo.scss:nombreDelArchivo.css
  4. YA que hayamos ejecutado esa linea se nos habrá creado otro archivo y se habrá fucionado ambos archivos, de esa manera ahora nosotros podemos realizar cambios en el archivo de SCSS y se vera reflejados
  5. Pero una vez que nosotros tengas una actualización, debemos de repetir todo el proceso por que no cuenta con auto-actualización, también tenemos una linea para la consola de comandos
  6. Y es la siguiente: sass --watch nombreDelArchivo.scss:nombreDelArchivo.css Y si por alguna razón deja de fallar esto debemos hacer lo siguiente
  7. Debemos de cerrar la orden con la consola de comando con control + C y volver a repetir el cógio sass --watch

Como crear variables en SASS

Las variables son muy utiles para poder nosotros hacer cambios de forma global, modificando un solo elemento que tengamo guardado, en el CSS PURO, lo hacemos tocando el ::ROOT y anteponiendo dos guiones(--) dejante del nombre de la variable, pero con SASS se nos simplifica mucho el trabajo, ya que solamente debemos de declararlas a nivel global y anteponerla con un signo de $, es decir quedaria de la siguiente manera:
$colorPrimary: #f2f2f2;

Como organizar nuestro proyecto si tenemos varios archivos

Para organizar varios archivos de un mismo proyecto con CSS PURO, teniamos dos opciones la primera hacer cuantos archivos fueran necesarios para aplicar los estilos y la otra manera es hacer los archivos y hacer un importanción directa en la hoja del CSS de esta manera queda una sola ruta de CSS carga en el HTML y todos los archivos quedan es dentro del CSS.
De esta última manera es que nosotros vamos a trabajar con las incorporaciones de varias hojas de estilos con SASS, y haremos lo siguiente:

  • Primero vamos a crear los archivos y le vamos anteponer en el nombre un guión bajo (_) es muy importante ponerselo, por que de esa manera es que va a interpretar que estamos creando una nueva hoja que no es primaria.
  • Luego iremos a la hoja primaria de SASS y haremos la importación de la hoja, esto se hace con la siguiente declaración @import "nombreDelArchivo
  • A la hora de haceer la importanción es importante colocar el nombreDelArchivo entre comillas dobles (" ") y sin el guión bajo, de esta manera ya tenemos la importanción lista.
